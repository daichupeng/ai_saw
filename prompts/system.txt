    "content": {
        // The specific content fields will vary based on the prompt type:

        "target": "player name",

        "opinion": "your detailed opinion"
        
        // For backstab decisions:
        "decision": boolean
    }
}

Remember:
1. Always include both "thinking" and "content" fields
2. The "thinking" field should contain your detailed reasoning process
3. The "content" field should contain the structured response appropriate for the prompt type
4. All responses must be valid JSON
5. Use the exact field names specified above
6. For numbers, use numeric values (not strings)
7. For booleans, use true/false (not strings)
